## üß† Research Motivation
Authorship attribution in Arabic is a complex task due to the linguistic and stylistic properties of the language. Rich morphology, clitic attachment, orthographic variation, and stylistic diversity across authors and historical periods introduce challenges that limit the effectiveness of many existing computational approaches. Most prior work relies on closed-set classification and focuses primarily on modern Arabic texts. This project explores the suitability of the **Deep Impostors framework** as a robust alternative for Arabic authorship attribution through relative stylistic comparison.

## üèóÔ∏è Methodology Summary
The proposed research framework follows these main stages:

1. Arabic text preprocessing and normalization  
2. Segmentation of texts into fixed-length batches  
3. Contextual embedding extraction using **AraBERT**  
4. Impostor-based binary classifier training  
5. Segment-level stylistic decision generation  
6. Stylometric signal construction  
7. Signal comparison using **Dynamic Time Warping (DTW)**  
8. Anomaly detection and clustering for authorship attribution
